POSTER
This poster was made through the use of Adobe Illustrator. The main purpose behind this piece was to design a vector illustration for an A4 poster series for MaltaDoors, a visual blog that documents the colourful and quirky facades and doorways on the Maltese Islands.

Beforehand, 25 preliminary photos were taken, each of a different door. Following so, a hand-drawn sketch was made before starting the vector process within Adobe Illustrator.

What you see here is the final result.
